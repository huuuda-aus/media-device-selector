# Contributing to Media Devices Selector

Thank you for your interest in contributing! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/media-devices-selector.git`
3. Install dependencies: `npm install`
4. Create a feature branch: `git checkout -b feature/your-feature-name`

## Development

- Run tests: `npm test`
- Build the project: `npm run build`
- Start development server: `npm run dev`

## Code Style

- Follow the existing code style
- Use TypeScript types where possible
- Write tests for new features
- Run `npm run format` before committing

## Submitting Changes

1. Make sure all tests pass
2. Update the documentation if needed
3. Commit your changes: `git commit -m "feat: your message"`
4. Push to your fork: `git push origin feature/your-feature-name`
5. Open a pull request

## Commit Message Format

```
type(scope): subject

[optional body]

[optional footer]
```

Example:
```
feat(hook): add device change detection

- Add event listener for device changes
- Update device list when devices change

Fixes #123
```

## Code of Conduct

Be respectful and inclusive. We follow the [Contributor Covenant](https://www.contributor-covenant.org/).
